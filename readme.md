# Bandcamp Art Downloader

```
usage: bc.py [-h] [--artists [ARTISTS [ARTISTS ...]]] [--albums [ALBUMS [ALBUMS ...]]]
             [--tracks [TRACKS [TRACKS ...]]]

optional arguments:
  -h, --help            show this help message and exit
  --artists [ARTISTS [ARTISTS ...]]
                        Usernames
  --albums [ALBUMS [ALBUMS ...]]
                        Album URLs
  --tracks [TRACKS [TRACKS ...]]
                        Track URLs
```
